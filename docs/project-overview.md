# TTB Label Verifier - Project Overview

## Description

A web application for verifying TTB (Alcohol and Tobacco Tax and Trade Bureau) labels against regulations.

## Architecture

- **Backend**: FastAPI-based REST API
- **Frontend**: React + TypeScript + Vite
- **Engines**: 
  - Local OCR (Tesseract/EasyOCR)
  - Multimodal LLM (Gemini/GPT)

## Features

- Upload label images for verification
- Extract label information using OCR or LLM
- Verify compliance with TTB regulations
- Display verification results and issues
